# E-commerce de Celulares - TechShop

Este proyecto es un sitio web de comercio electrónico diseñado para la venta de celulares y sus accesorios. Incluye una sección de inicio atractiva con contenido multimedia, catálogo de productos organizado, reseñas de clientes, un formulario de contacto post venta

## Estructura del Proyecto
- `index.html`: Contiene la estructura semántica 
- `style.css`: Archivo de estilos que maneja la tipografía, colores, diseño en Flexbox, Grid y adaptabilidad móvil.

## Tecnologías Utilizadas
- HTML5
- CSS3
- Google Fonts
- Formspree (para el envío de correos desde el formulario de contacto)
- Netlify
